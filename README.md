### Developer commands

*Git* 

```
git rm -r --cached .
git add . 
git commit -m "Untrack files in .gitignore" 
```

You can use the following command if you want to ignore an specific file. 

E.g: *node_modules*

```
git rm -r --cached ./node_modules
```
